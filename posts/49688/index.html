ECMAScript 2020（ES11）新特性详解

ECMAScript 2020（也称为 ES11）引入了一系列新特性，进一步增强了 JavaScript 的功能和灵活性。主要的新特性包括 Optional Chaining、Nullish Coalescing Operator、BigInt、Promise.allSettled()、globalThis、module import.meta、String.prototype.matchAll 和 WeakRefs。下面将详细介绍这些新特性，包括 API 说明、参数、返回值及用法示例。
